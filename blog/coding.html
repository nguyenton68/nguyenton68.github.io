Power of python with DFS problems

The following problems can be solved in very concise way in python: (compare to C++, we don’t have to declare variable type, which sometimes can be very lengthy.

  1. Permutation: list all permutations of given array with 1 digit (a=[1,2,3])
  2. Permutation with duplicate: same as #1, but return only unique answers. No duplicates (a=[1, 2, 2] –> [1,2,2], [2,1,2], [2,2,1])
  3. Combination: given number n and k, find combination of k numbers in n. (n=4, k=2 –> [1,2], [1,3], [1,4], [2,3], [2,4], [3,4])
  4. Combination sums: given array of ints [2, 3, 6, 7], and a target = 7. Find all combinations such that sum equal target. Each number can be used multiple times. Return [[7], [2, 2, 3]]
  5. Subset: given array of ints [1, 2, 3]. Return all unique subsets of the array. Return [ [3], [1], [2], [1, 2, 3], [1, 3], [1, 2], [2, 3]]
  6. Subset II: same as subset, but the array can have duplicates. Given [1, 2, 2], Return [[1], [2], [1, 2], [2, 2], [1, 2, 2]]
