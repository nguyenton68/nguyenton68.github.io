Bisect in Python

Posted by Nguyen Ton on October 27th, 2019

In this post, I'm going to explain the difference between bisect_left and bisect_right in python library.

At first, bisect is used to search to insert position of an element in a SORTED list. Suppose you are given list = [1, 2, 3, 4, 7], what should number 5 be inserted in this list (and after insert 5, this list still be sorted)?

They are the same if the element we are trying to look for is not in the list. If we are searching for position of number 5 in above example. Both bisect_left and bisect_right will output the same result, index = 4.

But if you are searching for number 4 (it's in the list), the bisect_left and bisect_right results will be difference. Bisect_left will give index = 3 (which is the position of this number in the list). Bisect_right will give index = 4 (which is to the right of itself).

Another equivalent of bisect.bisect_right is bisect.bisect. These two are the same thing.
